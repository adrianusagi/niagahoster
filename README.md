# niagahoster

Niagahoster full-stack developer test web


## Direction

clone repository from git
```
git clone https://github.com/adrianusagi/niagahoster.git
```

build using docker
```
docker-compose build
```

up into container
```
docker-compose up -d
```

web will available on browser using __localhost:8005__
```
localhost:8005
```
